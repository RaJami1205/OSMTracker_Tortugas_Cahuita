#  Proyecto OSMTracker – Demo para Registro de Tortugas en Cahuita

##  Descripción
Este proyecto es una **demo de OSMTracker**, creada para apoyar a los **cuidadores de tortugas en Cahuita** en sus tareas de registro y monitoreo en campo.

##  Objetivo
Facilitar la documentación de eventos relacionados con tortugas marinas (nidos, huellas, nacimientos y amenazas) mediante una herramienta simple y adaptable para uso en la playa.

##  Sobre OSMTracker
OSMTracker es una aplicación que permite registrar ubicaciones GPS, notas y evidencias (como fotos o audio).  
Esta demo adapta su funcionalidad al contexto del registro de tortugas.

##  Contenido del Repositorio
Proyecto en fase **demo / prototipo**, sujeto a cambios según pruebas de campo.

##  Agradecimientos
A los cuidadores y voluntarios de Cahuita por su labor en la conservación de las tortugas marinas.
